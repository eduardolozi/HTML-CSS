Tags semânticas estruturais

Div é uma tag conainer que não possui semântica
Header é uma tag para o cabeçalho
Nav é uma tag para menus de navegação
Main é uma tag para indicar que esse conteúdo é o mais importante da página (só pode ter uma main por página)
Article é uma tag para uma seção do site que seja independente (posts por exemplo)
Section é uma tag para indicar uma seção de conteúdo (seção dos posts por exemplo)
Aside é uma tag para indicar conteúdos em segundo plano (posts relacionados, propagandas)
Footer é uma tag para indicar o rodapé
Address é uma tag para indicar um local ou informações de contato

Tags de Linha e de Bloco

Uma tag de linha não vai fazer saltos de linha em cima e embaixo. Além disso, ela ocupa apenas o seu espaço. Oi eu sou outro span
Uma tag de bloco vai fazer saltos de linha em cima e embaixo. Além disso, ela ocupa toda a largura da sua linha.
